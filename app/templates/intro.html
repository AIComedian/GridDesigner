Welcome — this local web interface can launch the desktop Stick Figure Animation GUI that lives in the repository.

To open the desktop GUI, click the button below. The GUI opens as a separate window (Tkinter).

Open the small control page (status / launch)

Notes

  • The GUI is started in a separate process using the same Python interpreter running this server.
  • Make sure tkinter and required Python packages (Pillow, imageio) are available to that interpreter.
  • On headless machines (no display) the GUI will fail to start.
